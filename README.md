# threejs-lesson

## Download

 - download [three.js](https://github.com/mrdoob/three.js/archive/master.zip)
 - install [node](https://nodejs.org/en/download/)

## Getting started

    $ git clone https://github.com/jbjw/threejs-lesson.git
    $ cd threejs-lesson
    $ node express.js
